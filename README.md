A production-grade industrial control system designed to simulate, integrate, and monitor heterogeneous devices using the OPC-UA protocol. This project demonstrates real-time data acquisition, subscription-based communication, anomaly detection, and system diagnostics in a distributed environment.

---

## 🎯 Scope

This project focuses on building a complete industrial integration pipeline that reflects real-world control systems. It covers:

* Simulation of industrial devices (temperature, pressure, voltage sensors)
* OPC-UA server-client architecture for standardized communication
* Real-time data streaming using subscription mechanisms
* Integration of multiple system layers (device → server → client → UI)
* Fault detection and system diagnostics
* Monitoring and visualization through a live dashboard

The system is designed to replicate environments where reliability, scalability, and maintainability are critical.
